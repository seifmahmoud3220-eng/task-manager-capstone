# Task Manager Capstone Project

## 🚀 Overview
This is a Full Stack Task Manager application built as part of the IBM Full Stack Developer course.

It allows users to add, update, delete, and view tasks.

---

## 🧠 Technologies Used

### Frontend:
- HTML
- JavaScript
- CSS (basic styling)

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB
- Mongoose

---

## ⚙️ Features

- Add new tasks
- View all tasks
- Update task status (done / not done)
- Delete tasks

---

## 🏗️ Architecture

Frontend → REST API → Backend (Node.js) → MongoDB
